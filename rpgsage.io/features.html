RPG Sage:
Feature Guide

Quick Notes

RPG Sage has a game system neutral core engine. This comes in the form of integrated dialog, dice, and game table mechanics. You can use RPG Sage with any game system you want or without a game system entirely!

That said, I am a Pathfinder 2e player, and so I have built a number of Pathfinder 2e specific features into RPG Sage. I have also added a couple customizations for other systems and plan to do more in the future, as I get time and enough requests.
Happy Gaming!
