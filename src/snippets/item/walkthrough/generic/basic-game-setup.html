Basic Game Setup

Send these commands by posting them to the my-game-gm channel.
make sure you are in the #my-game-gm channel.

Create a game called: "My Game".
    sage!!game create name="My Game"

This example has Prefix sage, Action !!, Command game, SubCommand create, and Argument name with a value of My Game.

Update the Game Master channel to allow commands, admin commands, dice, and search for game masters while refusing players.
    sage!!channel set admin=true commands=true dice=true search=true gamemaster=write player=none

This example has Prefix sage, Action !!, Command channel, SubCommand set, and Arguments admin=true, commands=true, dice=true, search=true, gamemaster=write, and player=none.

Add the In Character channel to the game, allowing game masters and players to use dialog and dice.
    sage!!channel add #my-game-ic dialog=true dice=true gamemaster=write player=write

This example has Prefix sage, Action !!, Command channel, SubCommand add, and Arguments #my-game-ic, dialog=true, dice=true, gamemaster=write, and player=write.

Add the Out of Character channel to the game, allowing game masters and players access to all of the features.
    sage!!channel add #my-game-ooc admin=true commands=true dialog=true dice=true search=true gamemaster=write player=write

This example has Prefix sage, Action !!, Command channel, SubCommand add, and Arguments #my-game-ooc, admin=true, commands=true, dialog=true, dice=true, search=true, gamemaster=write, and player=write.

Add the Game Master to the game.
    sage!!gm add @UserMention

This example has Prefix sage, Action !!, Command gm, SubCommand add, and Argument @UserMention.

Add the Players to the game.
    sage!!player add @Player1Mention @Player2Mention @Player3Mention @Player4Mention

This example has Prefix sage, Action !!, Command player, SubCommand add, and Arguments @Player1Mention, @Player2Mention, @Player3Mention, @Player4Mention.
